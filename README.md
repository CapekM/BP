# Věnná města českých královen - API pro předávání grafických modelů

## Instalace

instalace závislostí

```sh
npm i
```

spuštění aplikace

```sh
npm start
```

pokud nemáte databázi, můžete si vytvořit v dockeru

```sh
docker-compose up -d
```

spuštění testů

```sh
npm test
```

spuštění testů s pokrytím kódu

```sh
npm run test-with-coverage
```
